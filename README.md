# EziTom Portfolio

A professional portfolio website featuring technical skills, projects, and a real-time contact system.

## Features
- **Modern UI**: Built with HTML, CSS, and vanilla JS with a focus on premium aesthetics.
- **Email Notifications**: Integrated with EmailJS for direct contact form enquiries.
- **Admin Dashboard**: LocalStorage-based messaging system to manage enquiries.
- **Responsive Design**: Fully optimized for mobile and desktop devices.

## Setup
1. Clone the repository.
2. Edit `config.js` to include your EmailJS Public Key, Service ID, and Template ID.
3. Open `index.html` in your browser.

## Technologies
- HTML5 / CSS3
- JavaScript (Vanilla)
- EmailJS SDK
- Lucide Icons
